4. Evidence of Learning

Featured Project: Face Recognition Door Lock System

A smart, automated access security prototype built using Python and OpenCV. By using a Haar Cascade Classifier and an LBPH algorithm, the application detects and verifies authorized users in real-time. Once validated, an instruction path routes to an Arduino microcontroller, manipulating a relay module to instantly operate a 12V physical solenoid lock. Unexpected parameters trigger integrated audio alarms.
